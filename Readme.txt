# Primer Challenge del curso Ciencia de Datos en Allura
##Consiste en realizar un anliais de rendimiento de 4 tiendas para detereminar cual es menos productiva
El analisis se basa en 5 pilares:
  1.Facturacion total de cada tienda
  2.Categorias mas polulares
  3.promedio de calificacion de los clientes
  4.productos mas y menos vendidos
  5.costo promedio del envio
Al final se presentara un informe que indique que tienda es la menos rentable para que el cliente este seguro de venderla
El entorno de trabajo sera un cuaderno interactivo de google collab basado en Jupyter, el lenguaje utilizado sera Phyton
